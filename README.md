# Person Counter

## Dependencies
1. Python v3
2. OpenCV v4.2.0
3. [tqdm](https://pypi.org/project/tqdm/)
4. [seaborn](https://seaborn.pydata.org/index.html)

## How to run?
```
python3 main.py
```

### The Graph
Since the images are numbered numerically, eg: `frame_0.jpg`, the X axis on the graph is the frame number.
